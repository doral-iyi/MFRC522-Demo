# MFRC522 Demonstration
## Purpose
To demonstrate the use of the MFRC522 RFID Reader & Writer.

## Bill-of-Materials
 * Arduino (pref. NodeMCU)
 * MFRC522 Sensor

## Getting Started
Refer to [Getting Started](getting-started/README.md).

## Uses
RFID is used in a multitude of cases. These include:

 * Security Systems
 * Attendance Systems
 * Location Tracking
 * Payment Systems
